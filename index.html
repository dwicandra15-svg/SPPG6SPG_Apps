Aplikasi MBG – SPPG

Sistem informasi pengelolaan program Makan Bergizi Gratis untuk Satuan Pelayanan Pemenuhan Gizi (SPPG). Memudahkan koordinasi antara penyedia dan sekolah penerima manfaat.

🔌 Koneksi Supabase
📋 Cara Menghubungkan Supabase:
1. Buat project di supabase.com
2. Buka Settings → API
3. Salin Project URL dan anon key
4. Paste di kolom di bawah ini
5. Setelah terhubung: username & password dapat diubah dan tersinkron otomatis
